# Future of ID — Module Excellence Standard

## Purpose

Every Future of ID capability module should function as a complete professional learning experience.

A strong module stands on its own while reinforcing the larger Future of ID philosophy: instructional designers are developing professional judgment, not simply consuming content about modern tools.

This document defines the quality standard for future module reviews. It should be used to evaluate whether a module creates a meaningful learner transformation, supports open entry, and helps learners practice future-ready instructional design.

## Seven Excellence Criteria

### 1. Professional Problem

What dilemma does the learner need to solve?

The module should begin from a recognizable professional problem, not a topic label. Learners should quickly understand the judgment they are being asked to develop.

### 2. Mental Model

What memorable idea should the learner leave with?

The module should give learners a durable way to think. This may come from a Learning Framework, a Situational Mental Model, or a concise module takeaway.

### 3. Design Practice

Does the learner exercise professional judgment?

The learner should make a meaningful design decision, evaluate a trade-off, or reason through a realistic instructional design problem. Practice should not feel like a knowledge check.

### 4. AI Design Review

Does AI strengthen reasoning rather than replace it?

AI should operate as a reviewer, coach, or critique partner. It should help learners sharpen judgment, test assumptions, and improve transfer without implying that AI owns the decision.

### 5. Reverse Build

Does the learner inspect professional reasoning?

Reverse Build should help learners reconstruct the experience from the instructional designer's perspective. It should reveal intentional decisions, instructional foundations, AI collaboration, trade-offs, and transfer value.

### 6. Reflection & Transfer

Does the learner connect the experience to future practice?

Reflection should help learners decide how the module changes their own work. It should move beyond personal reaction toward future design action.

### 7. Memorable Takeaway

What one sentence should remain six months later?

Every module should have one concise idea that can survive after the details fade.

## Quality Principles

- Every module should stand on its own.
- Every module should support AI interpretability.
- Every module should teach through transformation rather than explanation.
- Every module should reinforce professional judgment.
- Every module should avoid unnecessary complexity.
